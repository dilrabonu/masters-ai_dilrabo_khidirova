# Capstone Project Data Analysis Agent

## Project Overview
This is a data analysis agent built with Python and Streamlit, designed to provide insights from a given dataset.

## Requirements
- Python 3.8 - 3.12
- Streamlit
- Pandas
- Matplotlib
- OpenAI (optional)
